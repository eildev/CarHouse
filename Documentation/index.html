CarHouse React Template Documentation

Introduction

CarHouse - is a cross-browser compatible, retina-ready, and responsive landing page template with a single page and 9 sections using REACT technology. CarHouse is a visually appealing and elegant designed, interactive, readily adjustable, extremely modern, fast-loading, search engine optimized, effectively coded, well documented, bright, and completely responsive REACT template. Particularly helpful for car showrooms, new car companies, and websites that sell cars as well as mobile apps, blogs, and automotive websites.

Getting Started

In order to kickstart the development of your web src with CarHouse , the very first thing is to setup an React development environment.

Development Environment Setup

Name Description
index.js Used for configuration of project specific settings. You can add external styles and scripts, change the output folder, add assets, add environment files and more.
node_modules All external modules used are here. Do not mess with this folder, as it is auto-generated by using npm install.
package.json Contains all dependencies used for production and development.
src Contains all React Jsx code, assets and basically everything the end user will have access to.
tailwind.config.js You can configure this file if you need any design customization or use plugins.
vite.config.js If the plugin needs to be used, it must be added to the project's devDependencies and included in the plugin array of the vite.config.js configuration file.
  • Node.js and NPM : If NodeJS is not installed then download it from https://nodejs.org/. NPM comes bundled with Node.js

NPM Packages

  • Before proceding you'll need to install npm packages. You can do this by running npm install from the root of your project to install all the necessary dependencies.

Development Server

  • Run npm run dev for a dev server. Navigate to localhost:5173/. The src will autometically reload if you change any of the source files.

More Help

  • To get more help on the react use npm --help or go check out the React Support.

Fonts

By default, the template loads Inter font from Google Web Font Services & Karlais used as the title or bold font. You can change the font with the one that suits you best.

Font code can be found in the "src/main.css" file:


<link href="https://fonts.googleapis.com/css2?family=Karla:wght@200;300;400;500;600;700;800&display=swap" rel="stylesheet">


    <link href="https://fonts.googleapis.com/css2?family=Inter:wght@100;200;300;400;500;600;700;800;900&display=swap" rel="stylesheet">
